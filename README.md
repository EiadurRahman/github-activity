# GitHub User Activity CLI

This is a simple command-line interface (CLI) to fetch the recent activity of a GitHub user and display it in the terminal.

## Project Information

* **Project URL:** [https://roadmap.sh/projects/github-user-activity](https://roadmap.sh/projects/github-user-activity)
* **Challenge Source:** roadmap.sh

## How to Run

Follow these steps to run the project locally:

1. **Clone the repository:**
```bash
git https://github.com/EiadurRahman/github-activity.git
cd github-activity

```


2. **Install dependencies:**
```bash
pip install requests

```


3. **Execute the script:**
Run the program by passing a GitHub username as an argument:
```bash
python github-activity.py <username>

```
